# TSM商业秘密保护系统 - 更新日志

## V2.0.2 (2026-04-29)

### 新增功能

- **证书上传功能模块**
  - 新增上传证书模态框，支持完整的证书上传流程
  - 支持6种证书类型选择：著作权证书、商标证书、专利证书、软件著作权、ICP备案、其他证书
  - 支持拖拽上传和点击选择两种方式
  - 支持文件格式：PDF、JPG、PNG、DOC、DOCX
  - 单个文件大小限制：10MB
  - 最多支持上传5个文件
  - 实时上传进度展示
  - 上传成功/失败状态反馈
  - 表单字段：证书名称、颁发机构、证书编号、有效期

### 功能优化

- 首页快捷入口优化，点击可直接跳转至证书上传
- 证书管理页面按钮优化，"上传证书"按钮更醒目
- 页面交互反馈优化，增加Toast提示

### UI改进

- 保持简约蓝白信息图风格
- 线性扁平设计，白色工程图纸感背景
- 上传区域：虚线边框、拖拽图标、清晰的提示文字
- 证书类型选择采用卡片式布局，直观易选
- 模态框采用居中弹出，视觉层次清晰

### 交互流程

1. 点击"上传证书"按钮 -> 打开上传模态框
2. 选择证书类型（必填）
3. 填写证书信息（名称、颁发机构、编号、有效期）
4. 拖拽或点击上传证书文件
5. 查看上传进度和状态
6. 点击"确认上传"提交
7. 关闭模态框，返回证书列表
