# TSM商业秘密保护系统 - 联合信任

## 项目描述

TSM（Trusted Secrets Management）商业秘密保护系统是一个用于企业商业秘密认证和电子取证的可交互HTML原型系统。

## 访问地址

GitHub Pages: https://smknisyvecrix.github.io/tsm-prototype/

直接访问: https://smknisyvecrix.github.io/tsm-prototype/TSM原型_可交互.html
